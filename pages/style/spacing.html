Margin and Padding

Assign responsive-friendly margin or padding values to an element or a subset of its sides with shorthand classes. Includes support for individual properties, all properties, and vertical and horizontal properties.

  • m - for classes that set margin
  • p - for classes that set padding

Where sides is one of:

  • top - for classes that set margin-top or padding-top
  • bottom - for classes that set margin-bottom or padding-bottom
  • right - for classes that set margin-right or padding-right
  • left - for classes that set margin-left or padding-left

Margin Top

MÂ argin-top specifically sets the margin (space) at the top of an element. Margins are used to create space around elements, outside their borders.

The top margin controls the distance between the top of an element and the element above it (if any).

  • m-top - is the class name for margin-top

Use Cases of Margin

  • Margins create whitespace between elements like headers, paragraphs, and images, improving readability and visual clarity.
  • Margins adjust dynamically based on screen size using percentage-based or relative units.
  • Set consistent margins e.g mt-5 to ensure a professional look.
  • Center elements horizontally or vertically using margins.
  • Create margins between cards to improve spacing without cluttering the design.

Padding Top

Padding-top is used to define the padding (space) at the top of an element.

It increases the space inside the element, between the content (e.g., text or images) and the top edge of the elementâ€™s border.

  • p-top - is the class name padding-top

Use Cases of Padding

  • Add padding to text content for better readability by creating space around it
  • Increase padding around buttons or links to make them easier to tap or click.
  • Ensure consistent spacing inside containers like cards, banners, or forms.
  • Use padding to evenly distribute content within elements.
  • Add padding inside input fields to make text entry comfortable.
  • Prevent content from crowding borders by adding internal padding.
  • Use padding to evenly distribute content within elements.

Padding Bottom

Padding-bottom defines the space inside an element, specifically below its content, creating vertical space between the content and the element's bottom border.

Like other padding properties (top, left, right), it only affects the content inside the element, not the external layout or positioning.
