Images

The height and width properties are used to set the height and width of an element.

The height and width properties do not include padding, borders, or margins. It sets the height/width of the area inside the padding, border, and margin of the element..
  • .image-width-25 - The image takes 25% of the container
  • .image-width-50 - The image takes 50% of the container
  • .image-width-75 - The image takes 75% of the container
  • .image-width-100 - The image takes 100% of the container

Use Cases of Images

  • Ensures images resize proportionally within their containers.
  • max-width 100% prevents images from exceeding their container's width.
  • Image widths, such as 50% or 75% occupy only that percentage of the parent element's width.

Object Fit-Images

The object-fit property is used to specify how an image or video should be resized to fit its container.

This property tells the content to fill the container in a variety of ways; such as "preserve that aspect ratio" or "stretch up and take up as much space as possible".
  • Object fit-Cover - The image keeps its aspect ratio and fills the given dimension. The image will be clipped to fit.
  • Object fit-Contain - The image keeps its aspect ratio, but is resized to fit within the given dimension.
  • Object fit-Fill - This is default. The image is resized to fill the given dimension. If necessary, the image will be stretched or squished to fit.

Use Cases of Object Fit

  • Use Object fit to ensure media adapts well within custom-shaped containers
  • Object fit-fill scales placeholders or decorative images to fit unique container shapes.
  • Object fit-Contain displays images in consistent sizes without cropping.
  • Object fit-Cover display images that fill the container while maintaining image quality.
