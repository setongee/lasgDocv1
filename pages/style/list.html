List

A list is used to group a set of related items together, providing structure and organization to the content.

Unordered list

An unordered list is used to display a list of items where the order doesn't matter. Each item in the list is marked with a bullet (default), though this can be customized with CSS.

  • ul - is an abbreivation for unordered list

    • Use Cases of Unordered List

    • They can be used to display related items that don't follow a particular order.
    • Unordered lists are commonly used for navigation menus, where the order of links is not crucial.
    • Ensure consistent spacing inside containers like cards, banners, or forms.

    Ordered list

    An ordered list is used when the order of items is important. Each item in the list is numbered or lettered, depending on the format specified (numeric by default).

  • ol - is an abbreivation for ordered list

    • Use Cases of Ordered List

    • Use ordered lists for tutorials, guides, or procedures that must be followed in a specific sequence..
    • Use for items that need to be displayed in order of importance or rank.
    • Use to list events in chronological order, such as historical events or project milestones.
    • Use for step-by-step instructions to fill out forms or complete online applications.
    • Use to present steps in an algorithm or coding logic.
    • Use for debugging processes or troubleshooting guides.
