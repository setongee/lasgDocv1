Hero Header

A hero header (or hero section) is a prominent, visually striking area at the top of a webpage, often just below the navigation bar. It serves as the first thing visitors see when they land on the page, and its purpose is to capture attention and communicate the most important message or value proposition of the website.

  • hero-header - is the class name used for the hero text

Based on different styles, we have hero-header1 and hero-header2.

  • hero-header1 - is the class name used for the hero text
  • hero-header2 - is another class name used for a different hero text slightly different from the hero-header1

Hero Description

A hero description is the text that accompanies a hero header or hero section on a webpage, usually positioned near or below the main headline. Its purpose is to provide additional context or detail about the content or message being presented in the hero header. While the hero headline grabs attention, the hero description offers more specific information or clarifies the value proposition.

  • hero-desc - is the class name used for the hero description

Card Title

A card title is a heading or label typically used in a card-style layout, which is a popular design pattern in user interfaces. Cards are containers that group related content, such as an image, text, or actions, in a visually distinct manner. The card title serves as the primary heading or label for the content within the card and helps users quickly understand what the card represents or what itâ€™s about.

Typically found below the card title, the description helps clarify what the card is about, offering more information without overwhelming the user. Section Title

A section title is a heading used to label or introduce a distinct section of content within a webpage or document. It acts as a clear indicator of what the upcoming content is about, helping users navigate the page easily. Section titles are essential for organizing content, making it more scannable, and improving user experience by breaking up information into digestible chunks.

  • sec-title - is the class name used for the section-title

Section Description

A section description is a brief explanatory text placed beneath or near a section title on a webpage or in a document. It provides additional context or details about the content that follows the section title, helping users understand what they can expect in that particular section. The section description is especially useful in larger pieces of content, where the user may need guidance on what each section covers.

Modal Title

A modal title is the heading or label that appears at the top of a modal window (a small, overlaying popup that appears on top of a webpage or app interface). The modal title gives the user a clear understanding of what the modal is about, helping them quickly grasp the purpose or content of the modal before they engage with it.

  • mod-title - is the class name used for the modal title
