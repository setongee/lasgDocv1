LASG Web Templates Installation

This installation is required for the Lagos State web templates downloaded, Here's a simple installation guide for setting up your web template project!

1. Install Node.js & npm

React requires Node.js and npm (Node Package Manager) to work. Most systems already have npm installed if you have Node.js installed.

Download and install Node.js: Visit Node.js website and download the latest version. After installing, you can check if it's installed properly by running these commands in your terminal:

node -v npm -v

2. Navigate to Your Project Folder

After the project is downloaded, move into the template project directory:

3. Install Project Dependencies

npm install

You may need additional libraries depending on your project: Kindly visit the npmjs.com website to get acces to so free js packages.

4. Start the Development Server

Now, you can start the React development server by running:

npm run dev

This will launch your app in a local development server, usually accessible at http://localhost:3000.

5. Edit Your App

Open the src/App.js file to modify the main component.

Any changes you make will be reflected in real-time on the browser because of React's hot-reloading feature.

6. Build the Production Version

When you're ready to deploy your app, build a production-ready version by running:

npm run build

This will create an optimized build in a build folder, which you can deploy to platforms like Netlify, Vercel, or your preferred hosting provider.
